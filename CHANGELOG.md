## [1.0.0] — 2025-06-16

### Initial Release

First public release of Northline Icons — a minimal, outline-based SVG icon library
built for modern web applications, SaaS dashboards, and admin panels.

### Added

**77 SVG icons across 11 categories:**

- **Actions** (22) — `plus`, `check`, `x`, `copy`, `trash`, `pencil`, `grip-vertical`,
  `archive`, `search`, `upload`, `download`, `external-link`, `rotate-ccw`, `refresh-cw`,
  `arrow-up-down`, `share-2`, `tag`, `star`, `star-filled`, `sparkles`, `wand`, `dice`

- **Security** (19) — `lock`, `lock-open`, `eye`, `eye-off`, `key`, `key-round`,
  `fingerprint`, `shield-check`, `shield-alert`, `shield-x`, `shield-number`,
  `shield-ellipsis`, `badge-check`, `cloud-lock`, `signal-low`, `signal-medium`, `signal-high`,
  `log-in`, `log-out`

- **Navigation** (8) — `home`, `arrow-left`, `menu`, `more-horizontal`, `map-pin`,
  `list`, `grid-2x2`, `layout-grid`

- **Users** (6) — `user`, `users`, `house-user`, `id-card`, `passport`, `briefcase`

- **System** (6) — `settings`, `server`, `wifi`, `code`, `plane`, `film`

- **Files** (3) — `folder`, `database`, `file-lock`

- **Commerce** (3) — `credit-card`, `shopping-bag`, `landmark`

- **Communication** (3) — `mail`, `bell`, `globe`

- **Alerts** (3) — `info`, `circle-help`, `clock-alert`

- **Theme** (2) — `moon`, `sun`

- **Brands** (2) — `brand-shield`, `brand-lock`

### Design Specifications

- 24×24 grid, `viewBox="0 0 24 24"`
- 1.8px stroke width, `round` linecap and linejoin
- Outline style, `fill="none"`
- Lucide-compatible kebab-case naming convention

### Removed

- 3 duplicate/backup files excluded from production:
  `lock_logo_backup.svg`, `shield_logo.svg`, `shield_logo_backup.svg`
